Quick Usage Guide

Import tokens in your JavaScript/TypeScript:

import { sColorPurple500, sColorSuccess700 } from "@spexop/tokens";

Or use CSS variables:

background-color: var(--s-color-purple-500);

Click any color swatch to copy its hex value!
