Quick Usage Guide

Import tokens in your JavaScript/TypeScript:

import { sColorPurple500, sSpacing4, sShadowCard } from "@spexop/tokens";

Or use CSS variables:

background-color: var(--s-color-purple-500);

Click any token card to copy its value!
